# E-Commerce Sales & Customer Analytics (SQL-Centric)

## 📌 Proje Hakkında
Bu proje, Brezilya merkezli e-ticaret devi **Olist**'in verilerini kullanarak, bir **Junior Veri Analisti**'nin uçtan uca çözmesi gereken iş problemlerine odaklanır. 

**Hedef:** Veriyi ham halden alıp (ETL), PostgreSQL üzerinde modellemek, iş KPI'larını hesaplamak ve Power BI ile karar vericilere sunmaktır.

## 🛠️ Teknik Stack
- **Database:** PostgreSQL (SQL-First Approach)
- **Language:** Python (ETL & Data Cleaning), SQL (Window Functions, CTEs)
- **Visualization:** Power BI & Excel
- **Analytics:** RFM Segmentation, Cohort Analysis, Operational Performance

## 📊 Veri Modeli (ERD)
*Şu an veri setindeki 9 tablo (Orders, Customers, Payments, Products vb.) arasındaki ilişkisel yapı kurulmaktadır.*
